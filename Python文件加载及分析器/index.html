状态码分析

状态码包含很多信息，。例如

304.服务器收到客户端提交的请求参数，发现资源未变化，要求客户端使用浏览器的静态资源的缓存。

404，服务器找不到请求的资源

304占比大，说明静态资源的缓存效果显；404占比大，说明网站出现错误链接，或者有人尝试嗅探网站的资源。

如果400、500的占比突然增大，说明网站一定出现了错误。
